Draggable before/after wipe with a red handle — show off repair work. Color placeholders by default; pass `before`/`after` nodes (e.g. `<img>`) to use real media.

```jsx
<BeforeAfterSlider beforeLabel="Antes" afterLabel="Después" start={50} height={420} />
```

Drag or click anywhere on the track. Works with mouse and touch.
