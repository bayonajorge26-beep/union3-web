Dark service tile (icon chip + title + description) for the Unión 3 services grid; red top-rule and lift animate in on hover.

```jsx
<ServiceCard icon={<WrenchIcon />} title="Frenos"
  description="Pastillas, discos y revisión completa del sistema de frenado." />
```

Pass `icon` as an inline SVG node. Designed for a 3-up CSS grid with `gap: 16px`.
